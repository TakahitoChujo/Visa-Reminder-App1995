## 🔧 技術スタック

### フロントエンド
- **フレームワーク**: React Native 0.73 + Expo SDK 50
- **言語**: TypeScript 5.3（厳格モード）
- **状態管理**: Zustand 4.5
- **ストレージ**: AsyncStorage
- **通知**: Expo Notifications
- **日付処理**: date-fns 3.3
- **ID生成**: uuid 9.0

### バックエンド
- **ローカルDB**: SQLite
- **クラウドDB**: PostgreSQL
- **APIサーバー**: Node.js/Express
- **認証**: JWT（Access + Refresh）
- **通知**: Firebase Cloud Messaging (FCM)
- **暗号化**: AES-256-GCM
