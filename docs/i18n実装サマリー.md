# 多言語対応（i18n）実装サマリー

**実装日**: 2026-02-20
**対応言語**: 日本語（ja）、英語（en）
**ステータス**: Phase 1〜3 完了 / Phase 4（vi, zh）未着手

---

## 技術構成

| ライブラリ | 用途 |
|-----------|------|
| `i18next` + `react-i18next` | 翻訳エンジン・React統合 |
| `expo-localization` | デバイスロケール検出 |
| `date-fns/locale` | 日付フォーマットのロケール対応 |
| `AsyncStorage` | 言語設定の永続化（キー: `@app_language`） |

### 言語決定の優先順位

1. AsyncStorageに保存済みの設定
2. デバイスのロケール
3. `ja`（フォールバック）

## 使い方

### コンポーネント内

```tsx
import { useAppTranslation } from '../i18n/useAppTranslation';

function MyScreen() {
  const { t, formatDisplayDate, currentLanguage } = useAppTranslation(['home', 'common']);

  return (
    <Text>{t('home:card.registerNew')}</Text>
    <Text>{t('common:button.save')}</Text>
    <Text>{formatDisplayDate(new Date())}</Text>  // 2026年2月20日 / Feb 20, 2026
  );
}
```

### React外（サービス層）

```typescript
import i18n from '../i18n';

const title = i18n.t('notification:title.4months');
```

### 言語切替

```typescript
import { changeLanguage } from '../i18n';

await changeLanguage('en'); // AsyncStorageにも自動保存
```

## plan名前空間の設計（課金対応）

プラン関連テキストは `plan.json` に集約し、将来のRevenueCat連携を想定したパラメータ化を行っている。

### 補間パラメータ

| パラメータ | 用途 | 例 |
|-----------|------|-----|
| `{{price}}` | プラン料金（将来RevenueCatから取得） | `¥400` |
| `{{limit}}` | 無料プランの登録上限（`maxCards`から取得） | `1` |
| `{{count}}` | 現在の登録数 | `3` |
| `{{plan}}` | プラン名 | `Premium` |

### 使用例

```tsx
const { t } = useAppTranslation(['plan']);
const maxCards = useUserStore.getState().getMaxCards();

// "無料プランでは最大1件まで登録できます"
t('plan:limit.freeMax', { limit: maxCards });

// "月額 ¥400 で始める"
t('plan:premium.priceButton', { price: '¥400' });
```

### 将来の拡張

RevenueCat導入時は `price` パラメータを動的に差し替えるだけで対応可能:

```tsx
const price = revenueCatProduct.localizedPriceString; // "$3.99" etc.
t('plan:premium.priceButton', { price });
```

## 言語切替UI

SettingsScreen内に言語選択セクションを追加済み。現在の言語にチェックマーク表示。

対応言語は `SUPPORTED_LANGUAGES` で管理:

```typescript
// frontend/src/i18n/index.ts
export const SUPPORTED_LANGUAGES = {
  en: { label: 'English', nativeLabel: 'English' },
  ja: { label: '日本語', nativeLabel: '日本語' },
};
```

## Phase 4: 今後の言語追加手順

ベトナム語・中国語簡体字を追加する場合:

1. `locales/vi/` または `locales/zh/` を作成し、11ファイルを翻訳
2. `i18n/index.ts` にimportとresources登録を追加
3. `SUPPORTED_LANGUAGES` にエントリ追加
4. `useAppTranslation.ts` の `DATE_FNS_LOCALES` と `DATE_DISPLAY_FORMATS` にロケール追加
5. 入管用語の正確性をネイティブスピーカーに確認（重要）
