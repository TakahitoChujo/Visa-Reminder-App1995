## 1. ターゲットユーザーの現実

### 在留資格別ユーザー像

| 在留資格 | 年齢層 | 性別比 | 収入目安 | 価格感度 |
|---------|--------|--------|---------|---------|
| 技能実習 | 20-30代 | 男性多め | 15-20万円（手取り） | 非常に高い（仕送りあり） |
| 特定技能 | 20-30代 | 男性多め | 18-25万円 | 高い |
| 技人国（エンジニア等） | 20-40代 | やや男性 | 25-40万円 | 中程度 |
| 留学 | 18-30代 | 半々 | 5-15万円（バイト） | 非常に高い |
| 日本人の配偶者 | 幅広い | 女性やや多め | 世帯依存 | 低〜中 |
| 経営・管理 | 30-50代 | 男性多め | 30万円〜 | 低い |

### 重要な前提

- ユーザーの大半は手取り15-25万円層（技能実習・特定技能・留学が多数派）
- 母国への仕送りがあり、可処分所得が限られる
- スマホ代すら節約対象の層が多い
- サブスク疲れの時代に「もう1つ月額を増やす抵抗感」は大きい

## 2. 行動心理学分析書への現実チェック

### 価格について

| 分析書の提案 | 現実的な評価 |
|------------|------------|
| 月480円（スタンダード） | 技人国・経営管理層なら問題ないが、技能実習・留学生には毎月480円は重い |
| 月980円（プレミアム） | ほぼ誰も選ばない。この金額ならNetflixを選ぶ |
| CVR 12% | 楽観的。ニッチユーティリティアプリの現実は2-5% |

### 利用頻度の根本的問題

- ビザ更新は1年〜5年に1回
- アプリを日常的に開く理由がほぼない
- 「今月使ってないのに払ってる」感が強い
- **月額サブスクモデルとの相性が根本的に悪い**

## 3. 課金モデルの結論

### 決定事項: Phase制で進める

#### Phase 1: DLを増やす期間（最初の半年〜1年）

- **完全無料で出す（課金機能なし）**
- 機能制限なしで使ってもらう
- 良いレビューを集める（App Store評価4.5以上を目指す）
- 在留外国人コミュニティで地道に広める

**理由:**
- 課金要素があるだけで低評価レビューのリスクがある
- 「無料で全部使える」→ 口コミされやすい
- このアプリが本当に必要とされているか検証する段階

#### Phase 2: 収益化（DLが安定してから）

月100DL以上になったらユーザーの行動データを元に課金導入:

- どの機能がよく使われているか
- 何件登録している人が多いか
- 通知をどのタイミングで開いているか

**データを見てから課金ポイントを決める。** 行動心理学分析書のテクニックはこのPhaseでフル活用する。

## 4. DL獲得戦略

### このアプリの競合は「スマホのカレンダーにメモ」

アプリをわざわざ入れる理由 = **調べる手間を全部なくす**

| カレンダーメモ | このアプリ |
|--------------|-----------|
| 自分で期限を調べて入力 | 在留資格の種類を選ぶだけ |
| 通知は1回だけ | 4ヶ月前→3ヶ月前→1ヶ月前と段階通知 |
| 何を準備するかは自分で調べる | 資格別チェックリストが出る |
| 書類一覧は入管HPで探す | アプリ内で完結 |

### チャネル別戦略

#### 1. ASO（App Store最適化）— 最優先

多言語でキーワードを最適化:

| 言語 | キーワード例 |
|------|------------|
| 日本語 | ビザ更新 リマインダー / 在留期限 通知 / 在留カード 期限 |
| 英語 | japan visa renewal / residence card expiry / zairyu card reminder |
| ベトナム語 | gia hạn visa nhật / thẻ lưu trú nhật bản |
| 中国語 | 日本签证续签 / 在留卡到期提醒 |

- アプリ名に検索キーワードを含める
- スクリーンショットを多言語で用意
- アプリ説明文も多言語で記載

#### 2. 在留外国人コミュニティへの直接投下（コストゼロ）

| 国籍 | 主要SNS | 具体的な場所 |
|------|--------|-------------|
| ベトナム | Facebook | 「ベトナム人 日本 生活」系グループ |
| 中国 | WeChat / 小红书 | 在日华人系グループ |
| フィリピン | Facebook | Filipino community in Japan系 |
| ブラジル | Facebook / Instagram | Brasileiros no Japão系 |
| ネパール | Facebook | Nepali in Japan系 |

- 広告っぽくしない。ストーリー形式で
- その言語で投稿する
- スクリーンショット付きで「無料です」を強調

#### 3. 行政書士・企業との連携

- 行政書士: 顧客に更新時期を忘れられるのが悩み → 「無料アプリを勧めてください」
- 外国人雇用企業: 人事担当者に「従業員に入れさせてください」→ 一気に数十DL

#### 4. コンテンツSEO（中長期）

- LP（ランディングページ）を1枚作成
- 「技術・人文知識ビザの更新手続きまとめ」等のブログ記事
- 記事末尾にストアリンク

### 実行優先順位

```
すぐやる（開発と並行）:
  1. アプリ名・説明文の多言語最適化（ASO）
  2. Facebookの在留外国人グループ5-10個にポスト

リリース後1ヶ月以内:
  3. 行政書士 3-5人にコンタクト
  4. LP作成

リリース後3ヶ月以内:
  5. 企業への提案
  6. SEO記事
```

## 5. 開発方針への反映

### 多言語対応の優先度を上げる（決定事項）

DLに直結するため、課金機能より先に多言語対応を実装する。

**優先順位:**
1. 英語（必須・最優先）
2. ベトナム語（在留外国人1位）
3. 中国語 簡体字（在留外国人2位）

### 課金機能は後回し

Phase 1では課金機能を実装しない。開発リソースは以下に集中:
- 多言語対応
- コア機能の完成度
- UI/UXの品質
